# 🎭 动漫社交功能总览

欢迎来到动漫文档的社交互动中心！这里汇集了讨论区、推荐系统、榜单排行和好友对比功能。

## 📋 快速导航

| 功能 | 描述 | 链接 |
|------|------|------|
| 💬 讨论区 | 每部作品的独立讨论帖 | [进入讨论](../../discussions) |
| 🎯 智能推荐 | 根据观看历史推荐相似作品 | [查看推荐](./recommendations/README.md) |
| 🏆 榜单排行 | 用户投票产生的实时榜单 | [查看榜单](./rankings/README.md) |
| 👥 好友对比 | 对比你和朋友的观看记录 | [开始对比](./compare/README.md) |

## 🚀 快速开始

### 第一步：标记你的观看记录
打开 [90后经典动漫收藏大全（可标记版）](../90s-anime-collection-checklist.md)，将 `[ ]` 改为 `[x]` 标记你看过的作品。

### 第二步：参与讨论
点击任意作品名称旁的 💬 图标，进入该作品的GitHub Discussion讨论帖。

### 第三步：获取推荐
根据你已看的作品，查看 [推荐页面](./recommendations/README.md) 发现新番。

### 第四步：参与投票
在 [榜单页面](./rankings/README.md) 为你喜欢的作品投票。

## 📊 功能详解

### 💬 讨论区系统

**分类结构：**
- 📁 [作品讨论](../../discussions?discussions_q=category%3A%22%E4%BD%9C%E5%93%81%E8%AE%A8%E8%AE%BA%22) - 280+部作品的独立讨论
- 📁 [推荐求助](../../discussions?discussions_q=category%3A%22%E6%8E%A8%E8%8D%90%E6%B1%82%E5%8A%A9%22) - 求推荐、分享心得
- 📁 [榜单投票](../../discussions?discussions_q=category%3A%22%E6%A6%9C%E5%8D%95%E6%8A%95%E7%A5%A8%22) - 参与排行榜投票
- 📁 [综合交流](../../discussions?discussions_q=category%3A%22%E7%BB%BC%E5%90%88%E4%BA%A4%E6%B5%81%22) - 怀旧、闲聊

**如何参与：**
1. 点击任意作品名称旁的 💬 图标
2. 或直接进入 [Discussions](../../discussions) 页面
3. 发表你的看法、评分或推荐

### 🎯 推荐系统

**三种推荐方式：**

1. **基于类型推荐** - "我喜欢热血番"
   - 按热血、科幻、恋爱等类型分类推荐
   - [查看按类型推荐](./recommendations/by-type/)

2. **协同过滤推荐** - "看过X的人也喜欢Y"
   - 基于用户观看数据的智能推荐
   - [查看相似作品对](./recommendations/similar-pairs.md)

3. **年代推荐** - "重温90年代经典"
   - 按年代推荐同类型作品
   - [查看按年代推荐](./recommendations/by-era/)

### 🏆 榜单系统

**实时更新的排行榜：**

| 榜单名称 | 更新频率 | 描述 |
|----------|----------|------|
| [最受欢迎作品](./rankings/most-popular.md) | 每周 | 总票数最高的作品 |
| [最被低估神作](./rankings/underrated.md) | 每月 | 高分但讨论度低的作品 |
| [类型细分榜](./rankings/by-genre/) | 每周 | 各类型的人气排行 |
| [怀旧经典榜](./rankings/era-classics.md) | 每季 | 按年代划分的经典榜 |

**投票方式：**
- 在榜单页面点击作品旁的 👍 按钮
- 或在作品讨论帖中添加反应

### 👥 对比功能

**对比维度：**
- 共同看过的作品
- 相似度评分
- 口味分析
- 互补推荐

**使用方法：**
1. 复制 [对比模板](./compare/template.md)
2. 填写双方观看列表
3. 自动生成对比报告

## 🎨 使用示例

### 场景一：寻找相似作品
```
用户：刚看完《灌篮高手》，想看类似的热血运动番

操作：
1. 打开 docs/social/recommendations/by-type/sports.md
2. 查看"热血运动类推荐"
3. 发现《足球小将》《棋魂》《头文字D》
```

### 场景二：参与作品讨论
```
用户：想聊聊《千与千寻》的观后感

操作：
1. 在主文档找到《千与千寻》
2. 点击 💬 图标
3. 进入讨论帖发表评论
```

### 场景三：和朋友对比观看记录
```
用户：想知道和朋友有多少共同看过的作品

操作：
1. 打开 docs/social/compare/template.md
2. 复制模板并填写双方数据
3. 生成对比报告
```
