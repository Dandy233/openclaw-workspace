# 动漫作品扩展信息数据

本文件存储每部作品的详细信息，包括导演、制作公司、集数、观看平台和OST推荐。

## 数据结构说明

```yaml
作品名称:
  director: 导演/监督
  studio: 制作公司/工作室
  episodes: 集数/时长 (TV: XX集, 电影: XX分钟, OVA: X集)
  platforms:
    - platform: 平台名称
      type: 免费/付费/会员
      url: 链接
  ost:
    composer: 作曲家
    theme_songs:
      - name: 主题曲名称
        artist: 演唱者
    must_collect: 是否值得收藏原声 (是/否)
    highlights: 经典曲目
```
